# happy-trip


# Naming Convention
## kabab-case
* component name (when used in .vue file) <user-info> </user-info>

## PascalCase
* folder name
* file name
* enum name

## camelCase
* properity name
* function name

## CAPITAL_CASE
* mutations name (when used in store)

## Guidelines
https://vuejs.org/v2/style-guide/
